# Lab1

## Опис застосунку
Консольний застосунок, який перетворює підмножину markdown у html теги.

## Інструкція, як зібрати та запустити проект
Спочатку потрібно встановити node.js

Команда запуску парсера:
```
node "path\main.js" "path\file.md"
```
або 
```
node path/main.js path/file.md
```
